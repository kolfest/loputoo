uint8 WDT_setWatchdog ( uint32  period)

Configures watchdog timer wakeup period Uses VLO as a source clock, which ranges from 4kHz - 20 kHz

Acceptable input: 2 - 31536000 (typically 7.2 seconds - 1314 days)

minimal 2 periods (typically 2 * 3.6 seconds)

typical 3.6 seconds per period (atleast 15 periods)

4kHz = 8 second period 8kHz = 4 second period 12kHz = 2.66666 second period 20kHz = 1.66666 second period

WDT_ERR
